				Сервер:


команда для запуска: 		java18 -jar server-1.1.jar

Для начала работы необходимо ввести порт. 

Доступные команды можно узнать по команде help

Удалить коллекцию пользователей и элементов можно по команде clear_everything

Узнать список пользователей можно по команде see_users




				Клиент:

 
команда для запуска: 		java18 -jar client-1.1.jar 

Есть стандартный пользователь с логином admin и паролем admin.

Для начала работы необходимо ввести порт. Список команд можно узнать, введя "help".

У каждого пользователя должен быть уникальный логин. Пароль может быть длиной от нуля до 20 символов.

Редактировать элементы могут только их создатели.

Команду execute_script можно проверить по файлу test.txt в папке resources

Устали от работы и хотите посмотреть на котиков? Команда draw вам поможет! Введите её и увидите одного из 12 расслабляющих котиков) 

Хотите узнать своё тотемное животное ? Команда my_totem_animal именно для вас! Каждый пользователь (кроме админа) получает одно из семнадцати(!) тотемных животных с прикольной картинкой и уникальным описанием! 



				Описание Route:
id - генерируется автоматически 

name - не может быть пустой 

coordinates - не может быть налл 
	coordinate x - int, больше -836
	coordinate y - float, меньше 840

creationDate - генерируется автоматически 

location from - может быть null
	location from x - long
	location from y - double 
	location from name - строка

location to - не может быть null 
	location to x - long
	location to y - double 
	location to name - строка

distance - может быть null, float, больше 1

creatorLogin - генерируется автоматически
